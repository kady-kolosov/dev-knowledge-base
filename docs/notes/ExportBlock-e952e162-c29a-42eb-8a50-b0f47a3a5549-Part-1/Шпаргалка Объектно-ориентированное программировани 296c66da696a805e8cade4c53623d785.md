# Шпаргалка: Объектно-ориентированное программирование на Python

# Базовые понятия

## Класс

Класс — это шаблон для создания объектов, он объединяет в себе данные (атрибуты) и поведение (методы) этих объектов. Класс определяется ключевым словом `class` за которым следует имя класса в формате `CamelCase`, затем ставиться двоеточие `:` и начинается блок инструкций.

```python
class MyClass:  # Заголовок класса
    pass        # Тело класса
```

## Экземпляр класса (объект)

Экземпляр класса — это объект, созданный на основе класса. Экземпляр класса создается путем вызова класса `MyClass()`.

```python
class MyClass:
    pass

obj = MyClass()  # Экземпляр класса
```

# Атрибуты

## Атрибуты класса

Атрибуты класса — это переменные, заданные внутри класса вне методов. Они принадлежат самому классу и являются общими для всех экземпляров этого класса. Если изменить атрибут класса через класс, это изменение отразится на всех объектах, использующих его значение из класса.

### Работа с атрибутами класса

Обращение к атрибуту класса: `ИмяКласса.имя_атрибута` или `getattr(object, name, default=None)`

```python
class Employee:
    company = 'Google'
    position = 'Developer'

print(Employee.__dict__)
# {..., 'company': 'Google', 'position': 'Developer', ...}

# Обращение к атрибуту класса: ИмяКласса.имя_атрибута
print(Employee.company)                                  # Google
print(Employee.name)                                     # ❌ AttributeError

# Обращение к атрибуту класса: getattr(object, name, default=None)
print(getattr(Employee, 'position'))                     # Developer
print(getattr(Employee, 'name', 'Default value'))        # Default value
```

Проверка наличия атрибута в классе: `"имя_атрибута" in ИмяКласса.__dict__` или `hasattr(obj, name)`

```python
class Employee:
    company = 'Google'
    position = 'Developer'

print(Employee.__dict__)
# {..., 'company': 'Google', 'position': 'Developer', ...}

# Проверка наличия атрибута в классе: ИмяКласса.__dict__
print('company' in Employee.__dict__)  # True
print('name' in Employee.__dict__)     # False

# Проверка наличия атрибута в классе: hasattr(obj, name)
print(hasattr(Employee, 'company'))    # True
print(hasattr(Employee, 'name'))       # False
```

Создание атрибута класса: `ИмяКласса.имя_атрибута = значение` или `setattr(obj, name_attr, value)`

```python
class Employee:
    company = 'Google'
    position = 'Developer'

print(Employee.__dict__)
# {..., 'company': 'Google', 'position': 'Developer', ...}

# Создание атрибута класса: ИмяКласса.имя_атрибута = значение
Employee.name = 'Isabella'
print(Employee.__dict__)
# {'..., 'company': 'Google', 'position': 'Developer', ..., 'name': 'Isabella'}

# Создание атрибута класса: setattr(obj, name_attr, value)
setattr(Employee, 'name', 'Natasha')
print(Employee.__dict__)
# {'..., 'company': 'Google', 'position': 'Developer', ..., 'name': 'Natasha'}
```

Изменение атрибута класса: `ИмяКласса.имя_атрибута = значение` или `setattr(obj, name_attr, value)`

```python
class Employee:
    name = "Isabella"
    age = 30

print(Employee.name, Employee.age)  # Isabella 30

# Изменение атрибута класса: ИмяКласса.имя_атрибута = значение
Employee.name = "Natasha"
Employee.age = 20
print(Employee.name, Employee.age)  # Natasha 20

# Изменение атрибута класса: setattr(obj, name_attr, value)
setattr(Employee, "name", "Natasha")
setattr(Employee, "age", 20)
print(Employee.name, Employee.age)  # Natasha 20
```

Удаление атрибута класса: `del ИмяКласса.имя_атрибута` и `delattr(obj, name)`

```python
class Employee:
    name = "Natasha"
    surname = "Kolosova"
    phone_number = "+7 (977) 388-63-82"

# Удаление атрибута класса: del ИмяКласса.имя_атрибута
del Employee.phone_number
print(Employee.phone_number)  # ❌ AttributeError
print(Employee.__dict__)      # {..., 'name': 'Natasha', 'surname': 'Kolosova', ...}

# Удаление атрибута класса: delattr(obj, name)
delattr(Employee, "phone_number")
print(Employee.phone_number)  # ❌ AttributeError
print(Employee.__dict__)      # {..., 'name': 'Natasha', 'surname': 'Kolosova', ...}
```

## Атрибуты экземпляра

Атрибуты экземпляра — переменные, заданные у конкретного объекта, обычно внутри методов экземпляра через `self` в методе `__init__`. Они принадлежат отдельному экземпляру и хранят его индивидуальное состояние, независимое от других объектов.

### Работа с атрибутами экземпляра

Такие действия, как обращение к атрибуту экземпляра, проверка наличия атрибута у экземпляра, создание, изменение и удаление атрибута экземпляра, выполняются теми же операциями, что и для атрибутов класса.

```python
class Employee:
    company = 'Google'
    position = 'Developer'

id_123 = Employee()       # Создание экзепляра класса
print(id_123.__dict__)    # {}
id_123.name = 'Isabella'  # Создание атрибута экземпляра
print(id_123.__dict__)    # {'name': 'Isabella'}

id_456 = Employee()       # Создание экзепляра класса
print(id_456.__dict__)    # {}
id_456.name = 'Natasha'   # Создание атрибута экземпляра
print(id_456.__dict__)    # {'name': 'Natasha'}

# Обращение к атрибуту класса через экземпляр: имя_экземпляра.имя_атрибута_класса
print(id_123.company)     # Google — общий для всех экземпляров
print(id_456.company)     # Google - общий для всех экземпляров

# Обращение к атрибуту экземпляра: имя_экземпляра.имя_атрибута_экземпляра
print(id_123.name)        # Isabella - данные независимы
print(id_456.name)        # Natasha - данные независимы
```

# Методы

## Метод экземпляра: параметр `self`

Метод экземпляра — это метод, определённый в классе, который вызывается через экземпляр, получает первым аргументом `self` и работает с данными конкретного объекта.

- `self` — ссылка на экземпляр, через который вызван метод
- Python передаёт экземпляр в `self` автоматически при вызове через экземпляр
- Через `self` доступны атрибуты и методы экземпляра, класса и статические методы

```python
class Employee:
    company = "Google"

    def set_profile(self, name, position):
        self.name = name
        self.position = position

    def get_profile(self):
        print(self.name, self.position)  # Доступ через self к атрибутам экземпляра
        print(self.company)              # Доступ через self к атрибутам класса

emp1 = Employee()
print(emp1.__dict__)                     # {}

emp1.set_profile("Arkady", "Developer")
print(emp1.__dict__)                     # {'name': 'Arkady', 'position': 'Developer'}

emp1.get_profile()                       # Arkady Developer
                                         # Google
Employee.get_profile(emp1)               # Эквивалентно такому вызову emp1.get_profile()
```

## Инициализация экземпляра: метод `__init__`

`__init__` — это магический метод, который автоматически вызывается при создании объекта и задаёт его начальные атрибуты.

```python
class Employee:
    def __init__(self, name, position):
        self.name = name
        self.position = position

emp1 = Employee("Arkady", "Developer")
print(emp1.__dict__)  # {'name': 'Arkady', 'position': 'Developer'}
```

Помимо аргументов, атрибутам можно сразу задавать фиксированные значения — они будут одинаковы для каждого нового экземпляра.

```python
class Employee:
    def __init__(self, name):
        self.name = name
        self.is_active = True

emp1 = Employee("Arkady")
print(emp1.__dict__)  # {'name': 'Arkady', 'is_active': True}
```

Не должен ничего возвращать — явный `return` с любым значением, кроме `None`, вызовет `TypeError`.

```python
class Employee:
    def __init__(self, name) -> None:
        self.name = name
        return "Some value"

emp1 = Employee("Arkady")  # ❌ TypeError
```

## Метод класса: `@classmethod`

Метод класса — это метод, определённый в классе с декоратором `@classmethod`, который вызывается через класс и через экземпляр, получает первым аргументом `cls` и работает с данными класса.

- `cls` — ссылка на класс, через который вызван метод
- Python передаёт класс в `cls` автоматически при вызове и через класс, и через экземпляр
- Через `cls` доступны атрибуты, методы класса и статические методы

```python
class Employee:
    company = "Google"
    employee_count = 0

    @classmethod
    def get_company(cls):
        print(cls.company)          # Доступ через cls к атрибуту класса

    @classmethod
    def create(cls, name, position):
        cls.employee_count += 1     # Изменение атрибута класса через cls
        return cls(name, position)  # Создание экземпляра через cls

Employee.get_company()              # Google
emp1 = Employee.create("Arkady", "Developer")
print(Employee.employee_count)      # 1
emp1.get_company()                  # Google — вызов через экземпляр тоже работает
```

## Статический метод: `@staticmethod`

Статический метод — это метод, определённый в классе с декоратором `@staticmethod`, который вызывается через класс и через экземпляр и не получает первым аргументом ни `self`, ни `cls`.

- Не привязан ни к экземпляру, ни к классу — это обычная функция внутри класса
- Python ничего не передаёт автоматически при вызове
- Доступ к классу или экземпляру возможен только если они явно переданы аргументом

```python
class Employee:
    @staticmethod
    def validate_position(position):
        return position in ("Developer", "Manager", "Designer")

print(Employee.validate_position("Developer"))  # True
print(Employee.validate_position("Cook"))       # False

emp1 = Employee()
print(emp1.validate_position("Manager"))        # True — вызов через экземпляр тоже работает
```
